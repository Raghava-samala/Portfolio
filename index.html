Projects & Labs

SIEM Demonstration using ELK Stack and Filebeat in VirtualBox Environment

This project demonstrates a Security Information and Event Management (SIEM) setup using the ELK Stack (Elasticsearch, Logstash, Kibana) along with Filebeat for real-time log forwarding and monitoring. The environment is fully virtualized using Oracle VirtualBox, simulating a segmented network infrastructure with both public and private DMZ subnets.

View Project
    Key VMs:
    • Web Server VM (Public DMZ) – Generates logs.
    • ELK Stack VM (Private DMZ) – Receives and analyzes logs.
    • Filebeat Agent – Installed on the Web Server VM to forward logs to Logstash.
